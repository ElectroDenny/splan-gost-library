# splan-gost-library

Библиотека условных графических обозначений (УГО) электронных компонентов для выполнения схем по стандартам ЕСКД в программе **sPlan 8**.

На данный момент акцент сделан на выполнении электрических принципиальных схем (Э3), но при желании можно развить данную библиотеку для выполнения любых нужных вам схем по ЕСКД.

Все УГО выполнены в строгом соответствии с актуальными ГОСТами. Если вы обнаружили несоответствие стандартам, пожалуйста, создайте **Issue**.

# Установка библиотеки

## Способ 1: через Git

Чтобы установить данную библиотеку необходимо перейти к расположению sPlan (например, C:\Program Files\sPlan 8.0\), найти в нём папку `BIBO` и склонировать в неё этот репозиторий:
```bash
cd C:\Program Files\sPlan 8.0\
cd BIBO
git clone https://github.com/ElectroDenny/splan-gost-library.git
```

## Способ 2: скачать архив

Если у вас не установлен Git, вы можете скачать библиотеку в виде ZIP-архива:

1. Перейдите на страницу репозитория: https://github.com/ElectroDenny/splan-gost-library
2. Нажмите кнопку Code → Download ZIP.
<img width="auto" height="300" alt="image" src="https://github.com/user-attachments/assets/ba6e67bb-ee74-49d0-aa85-0f1ae9954ce2" />

3. Распакуйте содержимое архива в папку BIBO вашей установки sPlan. Важно, чтобы после распаковки структура папок выглядела так:
  ```
  BIBO/
  └── splan-gost-library/
      ├── Components/
      ├── Fonts/
      ├── Forms/
      └── README.md
  ```

## Компоненты

Для того чтобы установленная библиотека появилась в sPlan необходимо перезапустить программу.

Для вставки компонента на схему откройте нужную категорию в окне выбора компонентов слева и перетащите нужный элемент на лист. Все УГО для удобства разбиты в папки по категориям в соответствии с буквенными кодами элементов по ГОСТ 2.701-84.

<img width="1168" height="1039" alt="image" src="https://github.com/user-attachments/assets/f75a9ba7-1c85-4448-aeca-3aab60b1b971" />

## Формы

Папка Forms содержит готовые формы 1, 2, 2а, выполненные в соответсвии с ГОСТ Р 2.104-2023 &laquo;Основная надпись&raquo;.

Чтобы использовать форму:
1. В sPlan выберите "Формы" → "Загрузить форму из файла"

<img width="257" height="150" alt="image" src="https://github.com/user-attachments/assets/2f90d16f-7b6a-4a80-999b-f86949efc105" />

2. Укажите нужный .sbk файл из соответствующей подпапки `sPlan80/BIBO/splan-gost-library/Forms`.
3. Настройте соотвествующий выбранному в п.2 файлу формат и ориентацию листа.
