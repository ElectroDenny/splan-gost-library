# splan-gost-library

Библиотека условных графических обозначений (УГО) электронных компонентов для выполнения электрических принципиальных схем (Э3) по стандартам ЕСКД в программе **sPlan 8**.  
Все УГО выполнены в строгом соответствии с актуальными ГОСТами. Если вы обнаружили несоответствие стандартам, пожалуйста, создайте **Issue**.

# Установка библиотеки

## Способ 1: через Git

Чтобы установить данную библиотеку необходимо перейти к расположению sPlan (например, C:\Program Files\sPlan 8.0\), найти в нём папку `BIBO` и склонировать в неё этот репозиторий:
```bash
cd C:\Program Files\sPlan 8.0\
cd BIBO
git clone https://github.com/ElectroDenny/splan-gost-library.git
```

## Способ 2: скачать архив

Если у вас не установлен Git, вы можете скачать библиотеку в виде ZIP-архива:

1. Перейдите на страницу репозитория: https://github.com/ElectroDenny/splan-gost-library
2. Нажмите кнопку Code → Download ZIP.
3. Распакуйте содержимое архива в папку BIBO вашей установки sPlan. Важно, чтобы после распаковки структура папок выглядела так:
  ```
  BIBO/
  └── splan-gost-library/
      ├── Components/
      ├── Fonts/
      ├── Forms/
      └── README.md
  ```
4. После распаковки библиотека будет доступна в sPlan (может потребоваться перезапуск программы).
